## 🚀 GETTING STARTED (3 SIMPLE STEPS)

### Step 1: View Your Website (Already Running!)
```bash
# Visit in browser:
http://localhost:3000

# Or if server stopped, start it:
npm run dev
```

### Step 2: Configure Email (Choose One)
**Option A: Gmail (Easiest - 5 minutes)**
1. Go to: https://myaccount.google.com/apppasswords
2. Get 16-character App Password
3. Add to `.env.local`:
   ```
   SMTP_USER=your-email@gmail.com
   SMTP_PASSWORD=xxxx-xxxx-xxxx-xxxx
   ```

**Option B: Use Mailtrap (Testing - Free)**
1. Sign up: https://mailtrap.io
2. Get SMTP credentials
3. Add to `.env.local`

See `SETUP_GUIDE.md` for more options

### Step 3: Deploy (When Ready)
```bash
npm install -g vercel
vercel deploy
```
