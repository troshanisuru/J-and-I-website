## 📂 PROJECT STRUCTURE

```
src/
  ├── app/
  │   ├── page.tsx           ← Main landing page
  │   ├── layout.tsx         ← HTML layout
  │   ├── globals.css        ← Styles
  │   └── api/booking/route.ts ← Booking API
  │
  └── components/
      ├── Header.tsx         ← Navigation
      ├── Hero.tsx           ← Hero section
      ├── Services.tsx       ← 5 services
      ├── BookingForm.tsx    ← Booking form
      ├── Testimonials.tsx   ← Reviews (6)
      └── Footer.tsx         ← Footer

Configuration Files:
├── .env.local              ← Your config (create from .env.example)
├── tailwind.config.js      ← Colors & styles
├── package.json            ← Dependencies
├── tsconfig.json           ← TypeScript
└── next.config.js          ← Next.js config
```

## 📧 EMAIL SETUP CHOICES

### Option 1: Gmail (Easiest - Free)
1. Get App Password from Google
2. Add to .env.local
3. Done! ✅

### Option 2: SendGrid (Professional - $20/mo)
1. Create API key
2. Add to .env.local
3. Done! ✅

### Option 3: Outlook (Free)
1. Add credentials
2. Add to .env.local
3. Done! ✅

### Option 4: Test with Mailtrap (Free)
1. Sign up at mailtrap.io
2. Use SMTP credentials
3. View test emails in dashboard
