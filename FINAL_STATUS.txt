═══════════════════════════════════════════════════════════════════════════════
                      EMAIL CONFIGURATION (CHOOSE 1)
═══════════════════════════════════════════════════════════════════════════════

OPTION A: GMAIL (EASIEST - 5 MINUTES)
  1. Go to: https://myaccount.google.com/apppasswords
  2. Create App Password for "Mail"
  3. Copy 16-character password
  4. Edit .env.local:
     SMTP_USER=your-email@gmail.com
     SMTP_PASSWORD=your-16-char-password
  5. Restart server

OPTION B: MAILTRAP (FREE TESTING)
  1. Sign up: https://mailtrap.io
  2. Get SMTP credentials
  3. Use in .env.local
  4. View test emails in dashboard

OPTION C: SENDGRID (PROFESSIONAL)
  1. Sign up: https://sendgrid.com
  2. Create API key
  3. Update .env.local with credentials
  4. Ready for production

See SETUP_GUIDE.md for detailed instructions on all options.
